PWA Features

1. Pages can be cached when online and page contents are accessible while offline.

Turn off the internet and refresh this page, to see the page data and images persist.

2. While the device is offline, data entered can be stored locally in browser and Synced with server when the device goes online.

Turn off the internet and add a customer. Turn on the internet and refresh to see the data persist.

3. Installable to homescreen in mobile devices and desktop to provide app like experience.

4. Consumes less space than apps installed in mobile devices.

5. Chrome provides good support for PWA.In ios safari has limited support.

6. Provides push notification feature to the website.

7. In unsupported browsers it defaults to regular website.
